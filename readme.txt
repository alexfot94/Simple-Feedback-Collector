Project Title
Feedback Submission and Display App

 Short Description
A basic web application that allows users to submit feedback through a form. Submitted feedback is saved on the server using PHP and optionally displayed below the form for reference.

 How to Use It:

1. Install XAMPP (or a similar local server stack like MAMP/WAMP).
2. Start Apache via the XAMPP control panel.
3. Place the project folder inside the `htdocs` directory of XAMPP.  
   Example: `C:\xampp\htdocs\feedback-app\`
4. Open your browser and go to:  
   `http://localhost/feedback-app/`
5. Fill out the form and submit your feedback.
6. Your feedback will be saved to a `feedback.txt` file and shown below the form.

 Technologies Used:

- HTML – For the form structure and content layout
- CSS – For styling and responsive layout
- JavaScript – For client-side form validation
- PHP – For server-side form processing and saving feedback to a text file
